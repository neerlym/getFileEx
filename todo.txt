
Создать программу, которая выводит на экран все файлы заданного расширения
из указанного каталога. Расширение и каталог задаются как опции в командной строке.
Если в каталоге нет файлов с таким расширением, вывести "Файлов не найдено".

1. Модуль, который возвращает расширение из полного имени файла.

2. Модуль, получающий список файлов в заданном каталоге.

3. Модуль, выдющий список файлов с заданным расширением. Использует
предыдущие два модуля.

Если каталог при вызове не указан, заданный каталог считается текущим
(откуда был произведён вызов программы).

Если расширение при вызове не указано, выводим надпись "Не указано расширение".

Перед выводом списка файлов указать каталог и расширение, с которыми работает программа.
Список файлов вывести в виде таблицы со столбцами имя, дата изменения, размер;
отсортировать по имени.

Использовать библиотеку argparse для парсинга параметров командной строки.
Добавить исключение и его обработку при неверном количестве параметров.
Отформатировать вывод списка файлов.
